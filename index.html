Project Showcase

Hybrid Quantum-Classical Fraud Detection

Applying Variational Quantum Circuits (VQC) to solve the challenge of extreme data imbalance (0.17% fraud rate) in financial transaction data.

The Hybrid Pipeline Architecture

01. Classical Prep

Data Subsampling

Balanced subset created from 280k+ transactions. Features V1-V4 extracted and normalized to [0, 2π].

02. Quantum Map

Angle Encoding

Classical features mapped to qubit rotation angles ($R_Y$). Embeds data into high-dimensional Hilbert space.

03. VQC Kernel

Ansatz & Measure

4-Qubit circuit with trainable weights. Entangling layers create correlations. Pauli-Z measurement yields prediction.

04. Optimization

Hybrid Loop

Classical Adagrad optimizer updates quantum weights based on MSE Loss gradient. 50 Epochs.
